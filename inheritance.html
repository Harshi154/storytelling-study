Inheritance in Java

Inheritance in Java is a fundamental Object-Oriented Programming (OOP) mechanism that allows a new class (subclass) to acquire the fields and methods of an existing class (superclass). It establishes a parent-child relationship between classes, modeling an "IS-A" relationship (e.g., a Car is a Vehicle).

The syntax utilizes the extends keyword:

class Vehicle { // Superclass (Parent)
    int speed = 60;
}

class Car extends Vehicle { // Subclass (Child)
    int doors = 4;
}

Core Properties & Rules of Java Inheritance

  • Code Reusability: Subclasses reuse code from the superclass, eliminating redundant field and method definitions.
  • Method Overriding: A child class can override a parent method to provide a specialized implementation, driving runtime polymorphism.
  • The super Keyword: Subclasses use super to call parent constructors, access parent methods, or reference hidden variables.
  • Selective Access Modifiers: Fields marked public or protected are inherited; private members remain strictly encapsulated inside the parent class and are not directly accessible.
  • No Constructor Inheritance: Constructors are never inherited by subclasses, though the subclass constructor must invoke the parent constructor (either implicitly or explicitly via super()).
  • The Object Root: Every single class in Java automatically inherits from the java.lang.Object class if no other parent is specified.

Hierarchical Inheritance
  • Single Inheritance (Supported): One child class extends exactly one parent class.
  • Multilevel Inheritance (Supported): A chain of inheritance where a class extends a child class (e.g., Grandparent → Parent → Child).
  • Hierarchical Inheritance (Supported): Multiple distinct child classes extend a single parent class.
  • Multiple & Hybrid Inheritance (Unsupported via Classes): Java explicitly does not support a class extending more than one parent class to prevent compiler ambiguity (the "Diamond Problem"). These can only be achieved by implementing multiple Java Interfaces.
