📘 Elements of Basic HTML Structure

Let's understand what each HTML tag does and why it is important in creating a webpage.

1. <!DOCTYPE html>

The DOCTYPE declaration defines that the document is an HTML5 document. It must always be the first line of your HTML code.

📝 Important Note

Always write <!DOCTYPE html> at the beginning of every HTML document.

2. <html>...</html>

The <html> element is the root element of the webpage. Every other HTML element is placed inside it.

3. <head>...</head>

The <head> section stores meta-information about the webpage. This information is not displayed on the webpage.

  • Page Title
  • Meta Tags
  • CSS Links
  • JavaScript Files

4. <title>...</title>

The <title> tag sets the title of the webpage.

Where is it Visible? Description
Browser Tab Appears as the page title.
Search Engines Used as the main title in search results.

5. <body>...</body>

The <body> tag contains all the visible content of a webpage.

  • ✔ Headings
  • ✔ Paragraphs
  • ✔ Images
  • ✔ Hyperlinks
  • ✔ Tables
  • ✔ Lists
  • ✔ Forms

⚠ Important Note

There can be only ONE <body> element in an HTML document.

📌 Essential HTML Elements (Tags)

HTML elements are the building blocks of every webpage. Some of the most commonly used tags are:

Tag Purpose
<h1> to <h6> Headings
<p> Paragraph
<a> Hyperlink
<img> Image
<table> Table
<ul>, <ol> Lists

🔤 HTML is Not Case Sensitive

HTML tags are not case sensitive. For example:

<P>Hello</P>

is the same as

<p>Hello</p>

💡 Best Practice

Although HTML accepts both uppercase and lowercase tags, the recommends using lowercase tags. XHTML requires lowercase tags only.

💻 HTML <body> Example

<html>

<head>
  <title>Title of the document</title>
</head>

<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>

</html>

📖 Definition and Usage

The <body> tag defines the document body. It contains all the visible content including headings, paragraphs, images, hyperlinks, tables and lists.

🌍 Global Attributes

The <body> tag supports all Global Attributes like:

  • id
  • class
  • style
  • title
  • lang

⚡ Event Attributes

The <body> tag also supports Event Attributes such as:

  • onload → Executes when the page loads.
  • onresize → Executes when the browser window is resized.
  • onunload → Executes when the page is closed.
