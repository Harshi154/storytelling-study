📘 HTML Basics

Learn the fundamentals of HTML with colorful notes, examples, illustrations, and explanations.

🌐 What is HTML?

HTML stands for HyperText Markup Language

HTML is the standard markup language used to create webpages.

It provides the basic structure of every website.

📌 Basic Parts of an HTML Page

Every HTML document consists of the following basic elements.

📄 DOCTYPE

Declares the document type and HTML version.

🌍 HTML

Root element of every webpage.

⚙ HEAD

Stores metadata, title, CSS files and scripts.

🖥 BODY

Contains everything visible on the webpage.

🖼 HTML Page Structure

The following illustration shows the structure of a simple HTML document.

Tip: Every webpage starts with <!DOCTYPE html> followed by <html>, <head> and <body>.

💻 Example HTML Document

A simple HTML document looks like this.

<!DOCTYPE html> <html> <head> <title>My Web Page</title> </head> <body> <h1>My First Heading</h1> <p>My First Paragraph</p> </body> </html>

📖 Explanation of Each Element

<!DOCTYPE html>

Defines the document as an HTML5 document.

<html>

Root element of the webpage.

<head>

Contains metadata, title, external CSS and JavaScript files.

<body>

Contains all visible webpage content.

<h1>

Defines the largest heading.

<p>

Defines a paragraph.

📄 Understanding <!DOCTYPE html>

Every HTML document must begin with <!DOCTYPE html>.

Why is DOCTYPE Important?

  • ✅ Declares the document as an HTML5 document.
  • ✅ Helps browsers render the webpage correctly.
  • ✅ Appears only once at the top of the page.
  • ✅ It is not case-sensitive.
<!DOCTYPE html>

🧩 What is an HTML Element?

An HTML element consists of an opening tag, the content, and a closing tag.

Example: The paragraph below is a complete HTML element.
<p>Welcome to HTML</p>

💻 HTML Element Syntax

The general syntax of an HTML element is shown below.

<tagname> Content goes here... </tagname>

⭐ Example

<h1>My First Heading</h1> <p>This is my first paragraph.</p>

📌 Empty HTML Elements

Some HTML elements do not have any content. These are known as Empty Elements.

<br> <hr> <img> <input>
Important Note

Empty elements do not require a closing tag.

📋 HTML Quick Summary

HTML Tag Purpose
<html> Root element of the webpage.
<head> Contains metadata.
<title> Displays page title in browser tab.
<body> Contains visible webpage content.
<h1> Main heading.
<p> Paragraph.
<br> Line break.

💡 Remember

  • Always begin with <!DOCTYPE html>.
  • Every webpage has one <html> element.
  • The <head> contains page information.
  • The <body> contains everything visible.
  • HTML tags are enclosed inside angle brackets (< >).
  • Most HTML elements have both opening and closing tags.
