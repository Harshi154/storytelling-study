XML (Extensible Markup Language)

A Complete Overview of XML in Web Technologies

Introduction

XML (Extensible Markup Language) is a text-based markup language derived from Standard Generalized Markup Language (SGML). It is designed to store, organize, and transport data rather than display it.

XML is software and hardware independent, making it an ideal technology for sharing data between different applications and web systems.

Core Characteristics of XML

  • Extensible: Developers can create custom tags according to application needs.
  • Data-Focused: XML stores and transports data without defining its appearance.
  • Public Standard: XML is developed by the World Wide Web Consortium (W3C).
  • Platform Independent: XML files work across different operating systems and programming languages.

XML vs HTML

Feature XML HTML
Primary Purpose Stores and carries data Displays and formats data
Tags Custom user-defined tags Predefined tags
Case Sensitivity Case Sensitive Not Case Sensitive
Syntax Strictly well-formed More flexible
Whitespace Preserves whitespace Ignores extra spaces

Structure of an XML Document

Every XML document follows a hierarchical tree structure with one root element and multiple child elements.

<?xml version="1.0" encoding="UTF-8"?> <contact-info> <name>Tanmay Patil</name> <company>TutorialsPoint</company> <phone>(011) 123-4567</phone> </contact-info>

Explanation of XML Structure

  • XML Declaration: Specifies XML version and character encoding.
  • Root Element: The parent element that contains all other elements.
  • Child Elements: Custom tags that store the actual information.

Core XML Technologies

  • DTD (Document Type Definition): Defines the structure and rules of XML documents.
  • XML Schema (XSD): Provides advanced validation using XML syntax.
  • XML DOM: Allows JavaScript and other languages to access and modify XML elements programmatically.
  • XSLT: Transforms XML documents into HTML pages or other formats.

Summary

XML is one of the most important technologies for storing and exchanging data. Unlike HTML, which focuses on presentation, XML is designed to transport structured information using custom tags. XML is platform independent, extensible, and widely used in web services, APIs, configuration files, and data exchange between applications.
