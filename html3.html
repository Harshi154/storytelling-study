📘 HTML Syntax Rules

1. Opening & Closing Tags

Most HTML elements need both opening and closing tags.

Correct: <p>This is a paragraph.</p> Incorrect: <p>This is a paragraph.
📝 Always close your HTML elements.

2. Proper Tag Nesting

Close tags in reverse order.

Correct: <strong><em>Text</em></strong> Incorrect: <strong><em>Text</strong></em>
⚠ Incorrect nesting may break your webpage.

3. Empty Elements

These tags don't need closing tags.

TagPurpose
<br>Line Break
<img>Image
<hr>Horizontal Line

4. Attributes

Attributes belong inside the opening tag.

Correct: <a href="https://google.com">Click</a> Incorrect: <a>href="https://google.com">Click</a>

5. DOCTYPE

Every HTML5 page starts with:

<!DOCTYPE html>
📝 It must be the first line of the document.

6. Lowercase Tags

Use lowercase tag names.

Best: <div>Hello</div> Avoid: <DIV>Hello</DIV>

7. Basic HTML Structure

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello World</h1> </body> </html>
