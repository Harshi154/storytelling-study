HTML & XML Syntax Components

Tags, Elements, Attributes, and Comments form the core structural syntax of both HTML and XML. HTML uses predefined tags to display content in a web browser, whereas XML uses custom tags to store and transport structured data.

🏷️

1. Tags

Tags are labels enclosed within angle brackets (< >) that define the beginning and ending of a component.


Start Tag: Indicates where an element begins.

<p>


End Tag: Indicates where an element ends using a forward slash.

</p>


Self-Closing Tag: Used when no content exists inside an element.

<img />
📦

2. Elements

An element is the complete structure consisting of an opening tag, content, and a closing tag.

HTML Example:

<p>Hello World</p>
XML Example:

<price>29.99</price>
⚙️

3. Attributes

Attributes provide additional information about an element. They appear inside the opening tag using the format:

name="value"
HTML Example:

<a href="example">
XML Example:

<person gender="female">

Note: XML attribute values must always be enclosed in quotation marks.

💬

4. Comments

Comments are notes written inside the source code. They are ignored by browsers and XML parsers.

<!-- Your comment goes here -->

Rules:

  • Comments cannot be nested.
  • Do not place comments inside opening or closing tags.
  • Comments are used only for documentation.
