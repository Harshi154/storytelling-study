📘 Elements of Basic HTML Structure

Learn the basic HTML tags used to create every webpage.

<!DOCTYPE html>

Defines the document as HTML5. It should always be the first line.

Note: Always start every HTML page with this declaration.

<html>

The root element that contains the entire webpage.

<head>

Stores meta-information such as title, CSS links and metadata.

<title>

Displays the webpage title in the browser tab and search results.

<body>

Contains all the visible content like headings, paragraphs, images and links.

Warning: An HTML document can have only one <body> element.

Essential HTML Elements

TagPurpose
<h1>-<h6>Headings
<p>Paragraph
<a>Hyperlink
<img>Image
<table>Table
<ul>/<ol>Lists

HTML is Not Case Sensitive

<P>Hello</P>

is the same as

<p>Hello</p>

W3C recommends using lowercase tags for good coding practice.

HTML <body> Example

<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Welcome</h1>
  <p>Hello World!</p>
</body>
</html>

Definition & Usage

The <body> tag defines the document body and contains all webpage content.

Global Attributes

id, class, style, title, lang

Event Attributes

  • onload – Runs when the page loads.
  • onresize – Runs when the browser window resizes.
  • onunload – Runs when the page closes.
